  MevsU 

User Story
As a competitive person, I would like to be able to create an application where I can create my own competitions.
I want to have my information stored in the application's database.
I want to be able to name my competition.
I want other competitors to join my competition.
I want to set a time limit for my competition.
I want to set a unit parameter for my competitions.
I want to set a reward for whoever the winner is.
I want to see my current competitions.
I want to see other competitions outside of my groups.
